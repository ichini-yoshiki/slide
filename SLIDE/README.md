# SLIDE フォルダについて

ここに **プレゼンごとのデック（資料セット）** をフォルダ単位で並べます。

## レイアウト

例:

- `2026_04_16_PPP/` … 1本のスライド一式（`slides.md`、`slide.html`、画像など）

新しい回の資料を追加するときは、既存のフォルダをコピーして名前を変えるか、同じ構成のフォルダを新しく作ります。

## ビルド

リポジトリのルートで、対象デックのパスを渡します。

```powershell
cd <このリポジトリのルート>
.\.venv\Scripts\Activate.ps1
python build_slides.py SLIDE/2026_04_16_PPP
```

## 各デックの中身（目安）

| ファイル | 役割 |
|----------|------|
| `slides.md` | 編集する原稿 |
| `slide.html` | ブラウザで開く（ビルドで中身が更新される） |
| `_logo_frag.txt` | ロゴ用 HTML 断片 |
| `*.png` など | スライド用画像（`slides.md` の `image:` から参照） |

詳しい操作やトラブルシュートは、リポジトリ直下の `README.md` を参照してください。
